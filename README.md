
**Northwind Intelligence Dashboard 2026**

**Project Overview**

A professional, end-to-end Power BI Intelligence Dashboard developed using the classic Northwind Traders dataset. This project transforms raw operational data into actionable strategic insights, featuring a modern Glassmorphism-inspired UI and advanced DAX analytics.
The dashboard is designed as a multi-page "Executive Suite" that caters to different organizational roles, from C-level executives to warehouse managers.

**Key Features**

**Landing Page Interface**: A sleek, software-like entry point with glowing navigation buttons for seamless User Experience (UX).

**Strategic Sales Hub:** High-level revenue tracking with geographic mapping and seasonal trend analysis.

**Operational Inventory Audit**: Real-time stock value calculations and price sensitivity analysis.

**HR & Employee Performance**: Data-driven appraisal system ranking staff based on revenue and logistics efficiency.

**Customer Loyalty Module**: Identification of VIP clients and market shift tracking using Ribbon Charts.

**AI-Powered Admin Suite:** NLP-based "Q&A" box for ad-hoc reporting and Waterfall charts for financial variance auditing.

**Technical DAX Measures Included**

This project utilizes complex DAX (Data Analysis Expressions) to drive insights:

Total Sales (Net): SUMX calculation accounting for unit price, quantity, and discounts.

Inventory Value: Real-time capital tracking (PriceÃ—Stock).

Sales Variance: Month-over-Month (MoM) growth analysis using DATEADD and time-intelligence.

Shipping Efficiency: Calculating average delays using DATEDIFF between order and shipment dates.
